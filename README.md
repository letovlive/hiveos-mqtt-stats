**HiveOS MQTT statistic publisher.**

The package contains MQTT publisher for HiveOS miner.

To deploy the project:

```
cd ~
git clone 
chmod +x install.sh
chmod +x start.sh
chmod +x stop.sh
./install.sh
./start.sh
```

To stop publishing:
```
./stop.sh
```

To remove the project:
```
cd ~/hiveos-mqtt-stats
chmod +x uninstall.sh
./uninstall.sh
```
